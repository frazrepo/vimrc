dockerfile.vim
==============

Syntax highlighting for Dockerfiles

**Note**: this plugin is now a part of [docker][1].

Installation
------------

Via pathogen, the usual way...

Features
--------

The syntax highlighting includes:

* The directives (e.g. `FROM`)
* Strings
* Comments
