anderson.vim
============
Dark vim colorscheme based on colors from Wes Anderson films.

Installation
------------
Use the plugin manager of your choice, here is an example for Vim-Plug:

`Plug 'gilgigilgil/anderson.vim'`

`:PlugInstall`


To preview:

`:colorscheme anderson`
